# Test Automation Framework (Assignment 5)

## Описание
Этот проект представляет собой фреймворк автоматизированного тестирования для веб-приложения BlazeDemo.
Проект выполнен в рамках Задания 5 и включает в себя использование следующих технологий:

- **Java & Selenium WebDriver** - для управления браузером.
- **TestNG** - для управления тестами и аннотациями (`@BeforeClass`, `@Test` и т.д.).
- **Log4j2** - для логирования шагов выполнения.
- **Extent Reports** - для генерации HTML-отчетов с графиками.
- **Page Object Model** - архитектурный паттерн.

## Требования выполнены:
1. **TestNG Annotations**: Драйвер инициализируется в BaseTest, тесты чистые.
2. **Logging**: Логи пишутся в консоль при каждом действии.
3. **Reporting**:
    - Генерируется отчет `ExtentReport.html`.
    - При падении теста (Failure) автоматически делается **скриншот** и прикрепляется к отчету.

## Как запустить тесты
В терминале выполните команду:
mvn test

## Где найти результаты
После запуска тестов автоматически создаются:
1. **HTML Отчет**: `test-output/ExtentReport.html`
2. **Скриншоты ошибок**: `test-output/screenshots/` (или просто `screenshots/`)
